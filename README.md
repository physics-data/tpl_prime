# Prime

## 问题背景

就是把素数列出来，实在没故事可以编了。

## 问题描述

你需要编写一个 Python 程序完成素数。具体来说，你需要修改项目中的 `prime.py`，并完成以下功能：

1. 从标准读入读入一个整数。
2. 输出 ** 小于 ** 这个整数的所有素数，以单个空格分割。
